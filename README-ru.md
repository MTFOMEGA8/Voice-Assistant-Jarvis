
# Project Title/ru

ИИ-помощник, которым можно управлять с помощью голоса прямо на рабочем столе. На скриншоте ниже показана одна из голосовых команд — просто произнесите её в микрофон, а затем задайте любой вопрос или сделайте любой запрос (голосом). После этого помощник предоставит вам голосовой ответ на ваш запрос.

##  Key Features

-  **Преобразование речи в текст:** Безупречное преобразование устной речи в текстовые команды.
-  **Smart Brain:** Работает на базе API Groq (Llama 3) для ведения диалогов, похожих на человеческие.
-  **Безопасность превыше всего:** Использование переменных среды для обеспечения безопасности ключей API.
-  **Расширенное ведение журналов:** Специальный модуль ведения журналов для отслеживания поведения системы.
-  **Чистый код:** Модульная структура, соответствующая лучшим практикам, для удобства обслуживания.

 
## Tech Stack

**Клиент:** Python, API Groq, ведение журналов.


## How to run

1. Для начала необходимо скачать папку «J.A.R.V.I.S.».
2. Затем перейдите на сайт https://console.groq.com/keys, чтобы получить ключ API, необходимый для правильной работы помощника.
3. Откройте файл «env.example» и замените «your_key_here» на свой ключ API.
4. Переименуйте файл «env.example» в «.env».
5. Запустите скрипт через файл «main.py».
