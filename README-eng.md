# Project Title/eng

An AI assistant that you can control with your voice right on your desktop. The screenshot below shows one of its voice commands—simply say it into the microphone, and then ask any question or make any request (by voice). It will then provide you with a voice response to your request.


##  Key Features

-  **Voice-to-Text:** Converts spoken language into text commands seamlessly.
-  **Smart Brain:** Powered by Groq API (Llama 3) for human-like conversations.
-  **Security First:** Environment variables (.env) used to keep API keys safe.
-  **Advanced Logging:** Dedicated logging module to track system behavior.
-  **Clean Code:** Modular structure following best practices for easy maintenance.

## Tech Stack

**Client:** Python, Groq API, Logging.


## How to run

1. To get started, you need to download the “J.A.R.V.I.S.” folder.
2. Next, go to https://console.groq.com/keys to obtain an API key, which is required for the assistant to function properly.
3. Open the “env.example” file and replace “your_key_here” with your API key.
4. Rename “env.example” to “.env”.
5. Run the script via the “main.py” file.
