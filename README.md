# 🎙 J.A.R.V.I.S. (Voice Assistant 1.3) 

Персональный голосовой ассистент, вдохновленный Железным Человеком. Умеет распознавать голос, отвечать через Groq (Llama 3) и запоминать своего создателя.

## 🛠 Технологии:
- **Python 3.10+**
- **Groq API** 
- **SQLite** 
- **SpeechRecognition & pyttsx3** 

---

## 🚀 Текущие возможности:
- [x] Автоматическое приветствие по имени из БД.
- [x] Интеграция с ИИ Groq для умных ответов.
- [x] Система логов для отслеживания инцидентов.
- [x] Чистый код без лишних выводов в консоль.
